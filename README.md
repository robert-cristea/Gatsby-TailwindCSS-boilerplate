# Demo site for gatsby-theme-tailwindcss

A demo site which is using the theme in a very basic configuration.

## How to use this repo

1.  Clone the repo
2.  Install dependencies


    ```sh
    yarn
    ```

    or

    ```sh
    npm install
    ```

3. Start the demo site

   ```sh
   yarn start
   ```

   or

   ```sh
   npm start
   ```

4. Play with the demo site.
